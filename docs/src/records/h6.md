# H6レコード（票数・3連単）

## データ特性

- **配信タイミング**: レース確定後
- **KEY_FIELDS**: `meet_year`, `meet_date`, `racecourse_code`, `meet_round`, `meet_day`, `race_number`

## 特記事項

### データ提供について

- 発売のないレースについてはレコードを提供しない
- 2004年8月14日以降提供
