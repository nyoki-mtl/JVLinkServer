# レコード仕様

このセクションは JV-Link の wire レコード仕様（固定長仕様由来）を説明します。

## Python typed decoder の対応範囲

`Client.query_stored()` / `Client.query_realtime()` の typed 経路で、現行実装が直接デコードするレコードは以下です。

- `RA`（レース詳細）
- `SE`（馬毎レース情報）
- `HR`（払戻）
- `WH`（馬体重）
- `O1`（単勝・複勝・枠連オッズ）

それ以外のレコード種別は `UnknownRecord` として返ります。

## すべてのレコード仕様（wire）
