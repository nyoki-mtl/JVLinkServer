# O1レコード（単複枠オッズ）

## データ特性

- **配信タイミング**: 発売中～確定まで段階的に配信
- **KEY_FIELDS**: `meet_year`, `meet_date`, `racecourse_code`, `meet_round`, `meet_day`, `race_number`

## 注意事項

- 2004年8月13日以前のデータではオッズの最大値が異なる
  - 単勝オッズ: 99.9倍が設定できる最高値（通常は999.9倍）
  - 複勝オッズ: 99.9倍が設定できる最高値（通常は999.9倍）
  - 枠連オッズ: 999.9倍が設定できる最高値（通常は9999.9倍）
