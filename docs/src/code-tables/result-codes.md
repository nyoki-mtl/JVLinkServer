# 着順・異常関連コード表

## 2101.異常区分コード

競走中の異常を表すコード。

| コード | 説明 | 略名(2文字) | 欧字名 | 欧字略名 |
|--------|------|------------|--------|----------|
| 0 | 下記以外 または未設定・未整備時の初期値 | | | |
| 1 | 出走取消 | 取消 | SCRATCHED | S |
| 2 | 発走除外 | 発除 | EXCLUDED BY STARTERS | ES |
| 3 | 競走除外 | 競除 | EXCLUDED BY STEWARDS | ER |
| 4 | 競走中止 | 中止 | FALL TO FINISH | FF |
| 5 | 失格 | 失格 | DISQUALIFIED | DQ |
| 6 | 落馬再騎乗 | 再騎 | REMOUNT AFTER A CROPPER | RM |
| 7 | 降着 | 降着 | DISQUALIFIED AND PLACED | DQ&P |

## 2102.着差コード

着差を表すコード。

### 基本コード

| コード | 説明 | 略名(3文字) | 欧字名 |
|--------|------|------------|--------|
| \_\_\_ | 未設定・未整備時の初期値 | | |
| A\_\_ | アタマ | アタマ | HD |
| D\_\_ | 同着 | 同着 | DH |
| H\_\_ | ハナ | ハナ | NS |
| K\_\_ | クビ | クビ | NK |
| T\_\_ | 大差 | 大差 | DS |
| Z\_\_ | １０馬身 | １０ | １０ |

### 馬身差コード

| コード | 説明 | 略名(3文字) | 欧字名 |
|--------|------|------------|--------|
| \_12 | 1/2馬身 | 1/2 | 1/2 |
| \_14 | 1/4馬身 | 1/4 | 1/4 |
| \_34 | 3/4馬身 | 3/4 | 3/4 |
| 1\_\_ | １馬身 | １ | １ |
| 112 | １ 1/2馬身 | １ 1/2 | １ 1/2 |
| 114 | １ 1/4馬身 | １ 1/4 | １ 1/4 |
| 134 | １ 3/4馬身 | １ 3/4 | １ 3/4 |
| 2\_\_ | ２馬身 | ２ | ２ |
| ... | （同様のパターン） | | |

### 注記
- 1バイト目：着差単位(馬身)の整数部
- 2バイト目：分数の分子
- 3バイト目：分数の分母
- 地方競馬では「短クビ」「短アタマ」の場合も「クビ」「アタマ」コードを設定
