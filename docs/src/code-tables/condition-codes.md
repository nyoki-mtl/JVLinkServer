# 馬場・天候関連コード表

## 2010.馬場状態コード

馬場の状態を表すコード。

| コード | 状態 | 欧字名（芝） | 欧字名（ダート） |
|--------|------|------------|----------------|
| 0 | 未設定・未整備時の初期値 | | |
| 1 | 良 | Good to Firm | Standard |
| 2 | 稍重 | Good | Good |
| 3 | 重 | Yielding | Muddy |
| 4 | 不良 | Soft | Sloppy |

## 2011.天候コード

レース時の天候を表すコード。

| コード | 天候 | 欧字名 |
|--------|------|--------|
| 0 | 未設定・未整備時の初期値 | |
| 1 | 晴 | Fine |
| 2 | 曇 | Cloudy |
| 3 | 雨 | Rainy |
| 4 | 小雨 | Drizzle |
| 5 | 雪 | Snow |
| 6 | 小雪 | Light Snow |
