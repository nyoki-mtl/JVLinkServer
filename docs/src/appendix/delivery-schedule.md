# 配信スケジュール

JV-Linkのデータ配信タイミングの詳細です。

## 配信休止日

1. **毎週月曜日（火曜AM6:00まで）**: データ配信休止
2. **日曜日**: リアルタイムデータ以外は配信なし

## 蓄積系データの詳細提供タイミング

### 特別登録馬情報（TOKU）

- **日曜夕刻**: ハンデ発表前の情報（翌週土日分の特別レース）
- **月曜夕刻**: ハンデ発表後の確定情報（1年間保持）

### レース情報（RACE）の段階的更新

| 曜日 | 時間 | 内容 |
|------|------|------|
| 木曜 | 15:00頃 | 特定G1のみの出走馬名表 |
| 木曜 | 16:30頃 | 全レースの出走馬名表（枠番・馬番なし） |
| 木曜 | 20:00頃 | 招待騎手・調教師略称設定済み |
| 金・土 | 12:00頃 | 出馬表（枠番・馬番あり） |
| 月曜 | 14:00頃 | 確定成績 |

```python
# PyJVLinkでの取得例
# 1. 木曜15:00頃 - 特定G1のみの出走馬名表
await client.query_stored(
    dataspec="RACE",
    from_datetime="20240301000000",
    option=1,
)

# 2. 木曜16:30頃 - 全レースの出走馬名表（枠番・馬番なし）
# 3. 木曜20:00頃 - 招待騎手・調教師略称設定済み
# 4. 金土12:00頃 - 出馬表（枠番・馬番あり）
# 5. 月曜14:00頃 - 確定成績
```

### 差分更新データ（DIFF/DIFN、BLOD/BLDN）

- **月曜14:00頃**: 前回提供分からの変更・追加情報
- **木曜20:00頃**: 同上（週2回の更新）

これらは差分更新のため、データベースを最新状態に保つために必要です。

### 調教情報（SLOP/WOOD）

- **毎日不規則**: 天候や休日により変動
- **基本的に月曜はなし**: 休業日のため
- **トレセン別**: 美浦と栗東で調教時間が異なる

## リアルタイムデータの提供タイミング

### 速報オッズ（0B30-0B36）

- **金土日**: 随時更新（発売開始後）
- **金曜19:00〜**: PAT発売開始時（特定レース）
- **提供期間**: 1週間

### 速報レース情報（0B12、0B15）

段階的な情報更新:

1. 出走馬名表発表時点（木曜）
2. 出馬表発表時点（金土）
3. 払戻・3着まで確定時点
4. 払戻・5着まで確定時点
5. 払戻・全馬着順確定時点
6. コーナー通過順確定時点

```python
# 各段階での情報取得例
# 出走馬名表時点（データ区分=1）
race_key = "202401070511"
result = await client.query_realtime(dataspec="0B15", key=race_key)

# 確定成績時点（データ区分=4）
result = await client.query_realtime(dataspec="0B12", key=race_key)
```

### 速報開催情報（0B14、0B16）の重要な注意事項

騎手変更、出走取消などは**発表後に取り消される場合がある**ため、毎回最新データで完全置換することを推奨します。

```python
# 良い例：毎回最新データで上書き
meet_date = "20240107"
latest_result = await client.query_realtime(dataspec="0B14", key=meet_date)
latest_data = [envelope.record async for envelope in latest_result.records]
cache.clear()
cache.extend(latest_data)
```

## 補てん情報（TCOV/RCOV）の特殊性

### 非蓄積系ソフト用の補てん情報

- **TCOV**: 特別登録馬情報の関連マスタ・過去走
- **RCOV**: レース情報の関連マスタ・過去走

提供期間の注意:

- 次回同種データ提供まで（1週間程度）
- 一時的な補てん目的のため、長期保存は推奨されない

## 提供期間による分類

| データ種別 | 提供期間 | 備考 |
|----------|----------|------|
| 蓄積系（RACE、DIFF等） | 1年間 | 長期保存可能 |
| 速報系（0B11-0B17） | 1週間 | 短期利用のみ |
| 時系列オッズ（0B41-0B42） | 1年間 | 分析用途 |
| 補てん情報（TCOV、RCOV） | 次回提供まで | 一時的補てん |
| マスタ情報（HOSE、COMM） | 削除しない | 永続保存 |

祝日などで配信スケジュールが変更される場合があります。
