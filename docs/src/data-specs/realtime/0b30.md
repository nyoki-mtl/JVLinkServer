# 0B30 - 速報オッズ（全賭式）

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [O1](../../records/o1.md) | 単勝・複勝・枠連オッズ |
| [O2](../../records/o2.md) | 馬連オッズ |
| [O3](../../records/o3.md) | ワイドオッズ |
| [O4](../../records/o4.md) | 馬単オッズ |
| [O5](../../records/o5.md) | 3連複オッズ |
| [O6](../../records/o6.md) | 3連単オッズ |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 発売開始後 | リアルタイムで随時更新 |

## 提供期間

- **提供単位**: レース識別キー
- **保存期間**: 1週間

## 注意事項

1. **発売状況**: 発売のないレースではレコードが提供されない
2. **レース中止**: 発売済みのオッズは提供される（データ区分=9）
3. **データ保持**: 1週間経過後のデータは自動的に削除
