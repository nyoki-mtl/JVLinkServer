# 0B11 - 速報馬体重

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [WH](../../records/wh.md) | 馬体重 |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| レース発走約1時間前 | 馬体重が発表される毎に追加・更新 |

## 提供期間

- **提供単位**: 開催日単位（同一開催日の全開催場の全レース）またはレース毎
- **保存期間**: 1週間

## 注意事項

1. **発表タイミング**: レース発走の約1時間前に発表
2. **地方競馬**: 一部の競走では馬体重が発表されない場合あり
3. **データ保存期間**: 速報データは1週間のみ保存
4. **出走取消**: 出走取消の馬の馬体重は含まれない
