# 0B16 - 速報開催情報（指定）

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [WE](../../records/we.md) | 天候馬場状態 |
| [AV](../../records/av.md) | 出走取消・競走除外 |
| [JC](../../records/jc.md) | 騎手変更 |
| [TC](../../records/tc.md) | 発走時刻変更 |
| [CC](../../records/cc.md) | コース変更 |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 金土日の随時 | 開催情報が発生・変更する毎に更新 |

特定の変更情報のみを指定して効率的に取得可能です。

## 提供期間

- **提供単位**: 変更情報単位
- **保存期間**: 1週間

## リクエストキー

- `query_realtime(dataspec="0B16", key=...)`
- 形式: `TTYYYYMMDDJJRRNNNNNNNNNNNNNN`
- `key` は `stream_events()` / `JVWatchEvent()` の `event.param` をそのまま渡します
- 例: `WE20240107A20120240107112233`

## 注意事項

1. **指定方式**: 必要な情報のみを指定して取得
2. **効率性**: 0B14（一括）より効率的なデータ取得が可能
3. **更新頻度**: 変更発生時に随時更新
