# 0B13 - 速報タイム型データマイニング予想

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [TM](../../records/tm.md) | タイム型マイニング |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 前日予想（出馬表発表後） | 出走馬確定後の基本予想 |
| 当日予想（天候馬場発表後） | 天候・馬場状態を考慮した予想 |
| 直前予想（馬体重発表後） | 馬体重変化を考慮した最終予想 |

最新予想発表毎に上書き更新されます。

## 提供期間

- **提供単位**: 開催日単位（同一開催日の全開催場の全レース）またはレース毎
- **保存期間**: 1週間

## 注意事項

1. **予想の更新**: 最新の予想で上書きされるため、履歴保持は自前で必要
2. **評価値の解釈**: 評価値は相対的な指標で、絶対的な勝率ではない
3. **データの利用**: 予想はあくまで参考情報
4. **提供タイミング**: 各段階の予想は条件が揃い次第提供
