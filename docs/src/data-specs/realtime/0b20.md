# 0B20 - 速報票数（全賭式）

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [H1](../../records/h1.md) | 票数1（単勝・複勝・枠連・馬連・ワイド・馬単・3連複） |
| [H6](../../records/h6.md) | 票数6（3連単） |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 前日発売最終 | 前日の発売締切後 |
| 確定後 | レース確定後の最終票数 |

## 提供期間

- **提供単位**: レース識別キー
- **保存期間**: 1週間

## 注意事項

1. **発売状況**: 発売のないレースではレコードが提供されない
2. **レース中止**: 発売済みの票数は提供される
3. **データ保持**: 1週間経過後のデータは自動的に削除
