# 0B14 - 速報開催情報（一括）

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [WE](../../records/we.md) | 天候・馬場状態 |
| [AV](../../records/av.md) | 出走取消・競走除外 |
| [JC](../../records/jc.md) | 騎手変更 |
| [TC](../../records/tc.md) | 発走時刻変更 |
| [CC](../../records/cc.md) | コース変更 |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 金土日の随時 | 開催情報が発生・変更する毎に追加・更新・削除 |

含まれる情報:
- **103 天候馬場状態**: 各競馬場の天候と馬場状態
- **104 出走取消・競走除外**: 出走取消や競走除外の情報
- **105 騎手変更**: 騎手の変更情報
- **106 発走時刻変更**: レースの発走時刻変更
- **107 コース変更**: コースや距離の変更

## 提供期間

- **提供単位**: 開催日単位（同一開催日の全開催場）
- **保存期間**: 1週間

## リクエストキー

- `query_realtime(dataspec="0B14", key=...)`
- 形式: `YYYYMMDD`
- 例: `20240107`

## 注意事項

1. **データの取り消し**: 一度発表された変更が取り消される場合あり
2. **更新頻度**: 開催情報は随時更新される
3. **開催日単位**: データは開催日単位で提供
4. **データの完全性**: レース当日まで全ての変更情報が揃わない可能性
