## 注意事項

1. **オッズ有効性**: オッズ値が0の場合は無効値、0より大きい値のみを有効として扱う
2. **枠連特殊性**: 最大8枠のため出走頭数が少ない場合は使用されない枠あり、同枠（ゾロ目）配当は通常より低い
3. **複勝オッズ解釈**: 最小オッズは3着以内配当が最多の場合、最大オッズは最少頭数の場合
