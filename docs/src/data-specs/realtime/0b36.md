# 0B36 - 速報オッズ（3連単）

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 発売開始後 | リアルタイムで随時更新 |

更新段階: 中間、前日売最終、最終、確定

賭式の特徴:
- **組み合わせ数**: n×(n-1)×(n-2)（18頭なら4896通り）
- **データサイズ**: 最大

## 提供期間

- **提供単位**: レース識別キー
- **保存期間**: 1週間

## 注意事項

1. **データ量の考慮**: 最大4896通りの組み合わせでデータ量が膨大
2. **処理時間**: 大容量データのため取得・処理に時間を要する場合がある
3. **発売時期**: 他の賭式より発売開始が遅い場合がある
4. **リソース使用量**: メモリ使用量に注意が必要
