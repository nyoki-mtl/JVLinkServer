# 0B12 - 速報レース情報（成績確定後）

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [RA](../../records/ra.md) | レース詳細 |
| [SE](../../records/se.md) | 馬毎レース情報 |
| [HR](../../records/hr.md) | 払戻情報 |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| レース確定後（段階1） | 払戻・3着まで確定時点 |
| レース確定後（段階2） | 払戻・5着まで確定時点 |
| レース確定後（段階3） | 払戻・全馬着順確定時点 |
| レース確定後（段階4） | 払戻・全馬着順・コーナー通過順確定時点 |

各確定段階でレコードが追加されます（レース確定毎にレコード追加）。

## 提供期間

- **提供単位**: 開催日単位（同一開催日の全開催場の全レース）またはレース毎
- **保存期間**: 1週間

## 注意事項

1. **段階的な確定**: 同じレースに対して複数回レコードが追加されます
2. **データの重複**: 各確定段階で全データが含まれます（差分ではない）
3. **タイミング**: レース確定から数秒〜数分でデータ提供
4. **欠場・除外**: 出走取消や競走除外の馬は結果に含まれません
5. **最新データの利用**: 同一レースに複数レコードがある場合は、最新のものを使用してください
