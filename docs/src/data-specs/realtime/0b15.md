# 0B15 - 速報レース情報（出走馬名表～）

## 基本情報

- **データ種別**: リアルタイム系

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [RA](../../records/ra.md) | レース詳細 |
| [UM](../../records/um.md) | 出走馬 |
| [SE](../../records/se.md) | 馬毎レース情報（結果確定段階） |

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 出走馬名表時点（木曜日） | 枠番・馬番なしの出走馬情報 |
| 出馬表時点（金・土曜日） | 枠番・馬番、騎手・負担重量確定 |
| 払戻・3着まで確定時点 | 上位3着までの着順と払戻金 |
| 払戻・5着まで確定時点 | 5着までの着順確定 |
| 払戻・全馬着順確定時点 | 全出走馬の着順確定 |
| 払戻・全馬着順・コーナー通過順確定時点 | 完全なレース情報 |

各段階および変更情報提供時に更新されます。

## 提供期間

- **提供単位**: 開催日単位（同一開催日の全開催場の全レース）またはレース毎
- **保存期間**: 1週間

## 注意事項

1. **段階的更新**: 同じレースに対して複数の段階のデータが存在
2. **変更情報の反映**: 騎手変更などの変更情報が提供された場合、該当項目が即時更新
3. **データの完全性**: 出走馬名表段階では枠番・馬番が未確定
4. **0B12との違い**: 0B15は出走馬名表発表から全過程をカバー、0B12は成績確定後のデータのみ
