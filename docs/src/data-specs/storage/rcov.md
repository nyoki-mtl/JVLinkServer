# RCOV/RCVN - 非蓄積系ソフト用 補てん情報（レース情報補てん）

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）

## RCOVとRCVNの違い

2023年8月8日以降、拡張版RCVNではフィールドサイズが拡張されました。新規開発ではRCVNの使用を推奨します。

拡張内容：
- 繁殖登録番号のサイズ拡張
- 生産者コードのサイズ拡張  
- 生産者名のサイズ拡張

## データ更新タイミング

| 曜日 | 時間 | 内容 |
|------|------|------|
| 木曜 | 20:00頃 ※2 | レース情報用の補てん情報<br/>レース情報に関連したマスタや過去走の情報 |

※2 データの内容に誤り・訂正等があった場合、随時修正済みデータの再提供を行うことがあります。

## 提供期間

- **提供単位**: 週単位（土日分の全開催場の全レース）
- **保存期間**: 次回RCOV提供まで

## 注意事項

1. **専用データ**: 非蓄積系ソフト専用のデータ種別
2. **大量データ**: 一度に大量のデータが含まれるため処理時間がかかる場合あり
3. **補てん情報**: レース情報（RACE）に関連するマスタ情報や過去走情報を補完
4. **提供期間**: 次回のRCOV提供時に上書きされるため、必要に応じて保存
5. **修正提供**: データに誤りがあった場合は随時修正済みデータが再提供される
