# YSCH - 開催スケジュール

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）
  - `option=3`: セットアップデータ
  - `option=4`: ダイアログ無しセットアップデータ（初回のみダイアログ表示）

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [YS](../../records/ys.md) | 年間開催スケジュール |

## データ更新タイミング

| 曜日 | 時間 | 内容 |
|------|------|------|
| 年末 | - | 翌年の開催スケジュールの概要を提供 |
| 月曜 | 14:00頃 | 今週の開催情報に変更項目・開催中止・開催終了を反映<br/>来週の開催情報に詳細情報を反映 |

## 提供期間

- **提供単位**: 年単位
- **保存期間**: 1年間

## 注意事項

1. **スケジュール変更**: スケジュールは変更される可能性があり、月曜日に更新される
2. **中止・順延**: 天災等による開催中止・順延は月曜日の更新で反映される
3. **対象範囲**: 地方競馬・海外競馬の開催情報は含まれない
4. **年末提供**: 翌年の開催スケジュール概要は年末に提供される
5. **詳細情報**: 来週分の詳細情報は月曜日に反映される
