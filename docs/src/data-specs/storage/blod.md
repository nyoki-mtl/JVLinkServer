# BLOD/BLDN - 血統情報

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）
  - `option=3`: セットアップデータ
  - `option=4`: ダイアログ無しセットアップデータ（初回のみダイアログ表示）

## BLODとBLDNの違い

2023年8月8日以降、拡張版BLDNではフィールドサイズが拡張されました。新規開発ではBLDNの使用を推奨します。

拡張内容：
- 繁殖登録番号のサイズ拡張
- 生産者コードのサイズ拡張
- 生産者名のサイズ拡張

## データ更新タイミング

| 曜日 | 時間 | 内容 |
|------|------|------|
| 月曜 | 14:00頃 | 前回提供分から変更・追加のあった血統情報 |
| 木曜 | 20:00頃 | 前回提供分から変更・追加のあった血統情報 |

データベースを最新情報に保つための差分データが提供されます。

## 提供期間

- **提供単位**: 提供日単位（変更・追加のあった蓄積情報全て）
- **保存期間**: 1年間
- **セットアップデータ収録期間**:
  - HN: 1986年以降（繁殖馬情報）
  - SK: 1986年以降（産駒情報）
  - BT: 全件（系統情報）

## 注意事項

1. **文字コード変換**: 古いデータでは文字化けに注意が必要
2. **差分更新の管理**: データ区分の確認が重要（1=新規、2=更新、0=削除）
3. **繁殖登録番号**: 空白の場合があり、処理時にチェックが必要
4. **fromtime指定**: 終了日時の指定はできません（開始日時のみ）
5. **系統情報の更新**: 系統情報（BT）は頻繁に更新されないが、新系統追加時は要注意
