# SLOP - 坂路調教情報

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）
  - `option=3`: セットアップデータ
  - `option=4`: ダイアログ無しセットアップデータ（初回のみダイアログ表示）

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [HC](../../records/hc.md) | 坂路調教情報 |

## データ更新タイミング

| 曜日 | 時間 | 内容 |
|------|------|------|
| 毎日 | 不規則 | 坂路調教がある場合に提供<br/>天候や休日の関係で坂路調教がない場合もある<br/>基本的には月曜はなし<br/>美浦と栗東では調教時間が異なる<br/>季節や天候等によっても、調教時間が異なる |

## 提供期間

- **提供単位**: トレセン単位（日毎トレセン毎の全馬の坂路調教データ）
- **保存期間**: 1年間

## 注意事項

1. **対象範囲**: 坂路調教は美浦・栗東のみ（地方・外厩は含まれない）
2. **天候影響**: 天候や馬場状態により同じタイムでも価値が異なる
3. **馬具影響**: 調教時の馬具（ハミ、装具）により負荷が変わる
4. **コメント**: 調教師のコメントは主観的評価を含む
5. **調教時間**: 美浦と栗東では調教時間が異なり、季節や天候によっても変動する
6. **月曜日**: 基本的に月曜日は調教が行われない
