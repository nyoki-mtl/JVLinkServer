# TCOV/TCVN - 非蓄積系ソフト用 補てん情報（特別登録馬情報補てん）

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）

## TCOVとTCVNの違い

2023年8月8日以降、拡張版TCVNではフィールドサイズが拡張されました。新規開発ではTCVNの使用を推奨します。

拡張内容：
- 繁殖登録番号のサイズ拡張
- 生産者コードのサイズ拡張  
- 生産者名のサイズ拡張

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [UM](../../records/um.md) | 競走馬マスタ（全出走馬（登録馬）の競走馬情報） |
| [CH](../../records/ch.md) | 調教師マスタ（全出走馬（登録馬）の調教師情報） |
| [BR](../../records/br.md) | 生産者マスタ（全出走馬（登録馬）の生産者情報） |
| [BN](../../records/bn.md) | 馬主マスタ（全出走馬（登録馬）の馬主情報） |
| [RC](../../records/rc.md) | レコードマスタ（対象レースに関する最新のレコード情報） |
| [RA](../../records/ra.md) | レース詳細（全出走馬（登録馬）の全過去分のレース詳細） |
| [SE](../../records/se.md) | 馬毎レース情報（全出走馬（登録馬）の全過去分の馬毎レース情報） |

## データ更新タイミング

| 曜日 | 時間 | 内容 |
|------|------|------|
| 月曜 | 14:00頃 | 特別登録馬情報用の補てん情報<br/>特別登録馬情報に関連したマスタや過去走の情報 |

## 提供期間

- **提供単位**: 週単位（翌週土日分の全開催場の特別レース・前発表のあるG1レースを含む）
- **保存期間**: 次回TCOV提供まで

## 注意事項

1. **専用データ**: 非蓄積系ソフト専用のデータ種別
2. **大量データ**: 一度に大量のデータが含まれるため処理時間がかかる場合あり
3. **補てん情報**: 特別登録馬情報（TOKU）に関連するマスタ情報や過去走情報を補完
4. **提供期間**: 次回のTCOV提供時に上書きされるため、必要に応じて保存
