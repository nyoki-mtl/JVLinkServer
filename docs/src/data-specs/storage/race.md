# RACE - レース情報

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）
  - `option=2`: 今週データ  
  - `option=3`: セットアップデータ
  - `option=4`: ダイアログ無しセットアップデータ（初回のみダイアログ表示）

## データ更新タイミング

### 通常開催（土日）の場合

| 曜日 | 時間 | 内容 |
|------|------|------|
| 木曜 | 15:00頃 | 特定のG1レースがある場合のみ、そのG1レースの出走馬名表情報 |
| 木曜 | 16:30頃 | 出走馬名表（枠番・馬番なし、初招待騎手・調教師の略称設定なし） |
| 木曜 | 20:00頃 | 出走馬名表（枠番・馬番なし、初招待騎手・調教師の略称設定あり） |
| 金曜・土曜 | 12:00頃 | 出馬表発表（枠番・馬番確定） |
| 月曜 | 14:00頃 | 確定レース情報、確定重勝式(WIN5)情報 |
| 不定期 | 不定期 | レース情報に変更・修正がある場合に提供 |

### データ区分による更新段階

- **データ区分=1**: 出走馬名表
- **データ区分=2**: 出馬表  
- **データ区分=3**: 3着まで確定
- **データ区分=4**: 5着まで確定
- **データ区分=5**: 全馬着順確定
- **データ区分=6**: 全馬着順+コーナー通過順確定
- **データ区分=7**: 確定成績
- **データ区分=9**: レース中止
- **データ区分=A**: 地方競馬
- **データ区分=B**: 海外競馬

## 提供期間

- **提供単位**: 開催日単位（同一開催日の全開催場の全レース）
- **保存期間**: 1年間
- **セットアップデータ収録期間**:
  - RA, SE, HR, H1: 1986年以降
  - H6: 2004年8月以降
  - O1, O2: 1993年6月以降
  - O3: 1999年10月以降
  - O4, O5: 2002年6月以降
  - O6: 2004年8月以降
  - WF: 2011年4月以降
  - JG: 2011年6月以降

## 注意事項

1. **データ量**: RACEデータは非常に大きいため、期間を限定して取得することを推奨
2. **レコード順序**: レコードの順序は保証されません。キー項目でソートして使用してください
3. **地方競馬・海外レース**: データ区分=A（地方）、B（海外）として提供される場合があります
4. **レース中止**: データ区分=9でレース中止情報が提供されます
5. **特定G1の事前提供**: 一部のG1レースは木曜15:00頃に先行して提供されます
