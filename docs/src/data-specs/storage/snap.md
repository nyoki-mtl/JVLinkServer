# SNAP/SNPN - 出走時点情報

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）
  - `option=3`: セットアップデータ
  - `option=4`: ダイアログ無しセットアップデータ（初回のみダイアログ表示）

## SNAPとSNPNの違い

2023年8月8日以降、拡張版SNPNではフィールドサイズが拡張されました。新規開発ではSNPNの使用を推奨します。

拡張内容：
- 繁殖登録番号のサイズ拡張
- 生産者コードのサイズ拡張  
- 生産者名のサイズ拡張

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [CK](../../records/ck.md) | 着度数・出走数（競走馬） |
| JK | 着度数・出走数（騎手） |
| KC | 着度数・出走数（調教師） |
| BC | 着度数・出走数（馬主） |
| SC | 着度数・出走数（生産者） |

## データ更新タイミング

| 曜日 | 時間 | 内容 |
|------|------|------|
| 木曜 | 20:00頃 | 出走馬名表発表時点での競走馬、騎乗騎手、調教師、馬主、生産者の着度数情報 |
| 月曜 | 14:00頃 | 騎乗騎手に変更があった場合に提供（変更後の騎手の出走馬名表発表時点での情報に差替え） |

## 提供期間

- **提供単位**: 週単位
  - 木曜：土日分の全開催場の全レース
  - 月曜：騎手変更のあったレコードのみ
- **保存期間**: 1年間

## 注意事項

1. **出走馬名表発表時点**: 各種着度数は木曜日の出走馬名表発表時点での成績
2. **騎手変更時の更新**: 月曜日に騎手変更があった場合、該当騎手の情報が差し替えられる
3. **週単位の提供**: 土日開催分がまとめて木曜日に提供される
4. **着度数の計算基準**: 出走馬名表発表時点までの全成績が反映される
