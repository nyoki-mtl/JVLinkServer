# TOKU - 特別登録馬情報

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）
  - `option=2`: 今週データ
  - `option=3`: セットアップデータ
  - `option=4`: ダイアログ無しセットアップデータ（初回のみダイアログ表示）

## 含まれるレコード種別

| レコード種別 | 説明 |
|------------|------|
| [TK](../../records/tk.md) | 特別登録馬 |

## データ更新タイミング

| 曜日 | 時間 | 内容 |
|------|------|------|
| 日曜 | 夕刻 | ハンデ発表前の特別登録馬情報（翌週土日分） |
| 月曜 | 夕刻 | ハンデ発表後の特別登録馬情報（翌週土日分） |

### データ区分
- **データ区分=1**: 日曜日提供（ハンデ戦斤量未定）
- **データ区分=2**: 月曜日提供（ハンデ戦斤量確定）

## 提供期間

- **提供単位**: 週単位（翌週土日分の全開催場の特別レース・前発表のあるG1レースを含む）
- **保存期間**: 
  - 日曜提供分: ハンデ発表後提供まで
  - 月曜提供分: 1年間
- **セットアップデータ収録期間**: 利用可能（詳細期間は要確認）

## 注意事項

1. **空エントリー処理**: 300頭分の領域が確保されているが、実際の登録馬数は通常20-30頭程度。エントリー番号が0または馬名が空の場合は除外が必要
2. **特別レースとは**: 正式な競走名を持ち、通常より高額な賞金のレース。グレードレース（G1、G2、G3、リステッド）を含み、事前登録が必要
3. **グレードコード変遷**: 2007-2009年は国際格付けレースで異なるコードを使用
4. **補完データ**: 月曜日にTCOV/TCVN（特別登録補完データ）も配信、登録馬の詳細情報を取得可能
5. **fromtime指定**: 終了日時の指定はできません（開始日時のみ）
