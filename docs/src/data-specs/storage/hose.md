# HOSE/HOSN - 競走馬市場取引価格

## 基本情報

- **データ種別**: 蓄積系
- **利用可能オプション**: 
  - `option=1`: 蓄積系データ（通常はこちらを使用）
  - `option=3`: セットアップデータ
  - `option=4`: ダイアログ無しセットアップデータ（初回のみダイアログ表示）

## HOSEとHOSNの違い

2023年8月8日以降、拡張版HOSNではフィールドサイズが拡張されました。新規開発ではHOSNの使用を推奨します。

拡張内容：
- 繁殖登録番号のサイズ拡張
- 生産者コードのサイズ拡張
- 生産者名のサイズ拡張

## データ更新タイミング

| タイミング | 内容 |
|----------|------|
| 不定期 | 市場終了後、概ね1ヶ月以内に提供 |
| 不定期 | 市場取引時に血統登録番号未確定の競走馬については、血統登録番号確定時に提供 |

## 提供期間

- **提供単位**: 提供日単位（変更・追加のあった蓄積情報全て）
- **保存期間**: 削除しない（永続的に保持）
- **セットアップデータ収録期間**: 1997年以降（競走馬市場取引価格情報）

## 注意事項

1. **データ保持期間**: 永続的に保持、1997年以降の全取引データが蓄積
2. **血統登録番号の確定**: 取引時点で未確定の馬は番号確定後にデータ提供
3. **価格情報**: 売却価格は円単位、主取りの場合は価格が0または未設定
4. **市場の種類**: 全国の競走馬セリ市、JRAブリーズアップセール、地方競馬関連セリ市を含む
5. **fromtime指定**: 終了日時の指定はできません（開始日時のみ）
